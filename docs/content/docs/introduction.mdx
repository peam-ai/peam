# Introduction to Peam

Peam is an intelligent AI-powered search and chat system that helps users find information on your website naturally. Instead of traditional keyword search, Peam understands context and provides conversational answers based on your content.

## What is Peam?

Peam provides:

- **AI-Powered Search**: Natural language understanding for better search results
- **Conversational Interface**: Users can ask questions and get immediate answers
- **Easy Integration**: Simple installation with minimal configuration
- **Content Indexing**: Automatically extracts and indexes your website content
- **Customizable**: Flexible API to match your brand and requirements

## How It Works

1. **Index**: Peam scans your built website and creates an optimized search index
2. **Query**: Users ask questions through a chat interface
3. **Search**: Peam finds relevant content using semantic search
4. **Answer**: AI generates contextual answers based on your content

## Key Features

- Built for modern web frameworks (Next.js, Vite, Astro, etc.)
- Uses state-of-the-art AI models (OpenAI, Anthropic, etc.)
- Lightweight client component
- Server-side processing for security
- TypeScript support

## When to Use Peam

Peam is perfect for:

- Documentation sites
- Knowledge bases
- Content-heavy websites
- E-commerce sites with product catalogs
- Any site where users need to find information quickly
