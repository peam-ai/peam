## Installation

Install Peam and required AI SDK packages:

```bash
npm install peam @ai-sdk/openai
```

## Setup

### 1. Add Environment Variables

Create a `.env.local` file in your project root:

```bash
OPENAI_API_KEY=your_openai_api_key_here
```

### 2. Add the AskAI Component

In your root layout (`app/layout.tsx`):

```tsx
import { AskAI } from 'peam/client';

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en">
      <body>
        {children}
        <AskAI />
      </body>
    </html>
  );
}
```

### 3. Create API Route

Create `app/api/peam/route.ts`:

```typescript
import { createHandler } from 'peam/server';
import { openai } from '@ai-sdk/openai';

export const POST = createHandler({
  model: openai('gpt-4o'),
  getSearchEngine: async () => {
    // Your search engine implementation
    // This will be provided by your indexed content
    return mySearchEngine;
  },
});
```

### 4. Index Your Site

After building your Next.js site, index it:

```bash
# Build your site
npm run build

# Index the output
npx peam ./out
```

For Next.js static export, make sure you have `output: 'export'` in your `next.config.js`.

## Configuration

You can customize the AskAI component appearance and behavior:

```tsx
<AskAI 
  placeholder="Ask anything about our docs..."
  theme="dark"
/>
```

See the [API Reference](/docs/api-reference) for all available options.
