# Overview

Inline layout that composes `AskAI.Header`, `AskAI.Messages`, `AskAI.Suggestions`, and `AskAI.Input`.

## Usage

```tsx
import { AskAI } from 'peam/client';

export default function Page() {
  return (
    <AskAI>
      <AskAI.Inline />
    </AskAI>
  );
}
```

## Notes

- Must be rendered inside `AskAI` or `AskAIProvider`.
