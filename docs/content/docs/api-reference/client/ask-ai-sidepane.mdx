# Overview

Sidepane chat surface that slides in from the right on desktop and from the bottom on mobile.

## Usage

```tsx
import { AskAI } from 'peam/client';

export default function Page() {
  return (
    <AskAI>
      <AskAI.Trigger />
      <AskAI.Sidepane />
    </AskAI>
  );
}
```

## Notes

- `AskAI.Sidepane` must be rendered inside `AskAI`.
- On desktop, opening the sidepane temporarily applies a right margin to the document body.
