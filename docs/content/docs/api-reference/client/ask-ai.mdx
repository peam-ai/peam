# AskAI

The main AI chat component with multiple display modes.

## Type Definition

```ts
import type { ReactNode } from 'react';
import type { AskAIChatProps, AskAIDialogProps, AskAISidepaneProps } from 'peam/client';

export type AskAIType = 'chat' | 'dialog' | 'sidepane';

export interface AskAIBaseProps {
  /**
   * API endpoint for handling AI requests.
   * @default '/api/peam'
   */
  endpoint?: string;

  /**
   * Array of suggested prompts to display to the user.
   */
  suggestedPrompts?: string[];

  /**
   * Maximum number of messages to keep in context before summarizing.
   * @default 10
   */
  maxMessages?: number;

  /**
   * Render the button inline instead of as a floating button.
   * @default false
   */
  inlineButton?: boolean;

  /**
   * Variant of the default button.
   * @default 'iconLabel'
   */
  buttonVariant?: 'icon' | 'iconLabel';

  /**
   * Additional CSS classes to apply to the root container.
   */
  className?: string;

  /**
   * Custom button component.
   */
  button?: (props: { isOpen: boolean; toggle: () => void; open: () => void; close: () => void }) => ReactNode;
}

export type AskAIProps =
  | ({ type?: 'chat' } & AskAIChatProps)
  | ({ type: 'dialog' } & AskAIDialogProps)
  | ({ type: 'sidepane' } & AskAISidepaneProps);
```

## Usage

### Chat Mode (Default)

```tsx
import { AskAI } from 'peam/client';

export default function Page() {
  return <AskAI type="chat" />;
}
```

### Dialog Mode

```tsx
import { AskAI } from 'peam/client';

export default function Page() {
  return <AskAI type="dialog" />;
}
```

### Sidepane Mode

```tsx
import { AskAI } from 'peam/client';

export default function Page() {
  return <AskAI type="sidepane" />;
}
```

### With Custom Button

```tsx
import { AskAI } from 'peam/client';
import { Button } from '@/components/ui/button';

export default function Page() {
  return (
    <AskAI
      type="dialog"
      button={({ isOpen, toggle }) => <Button onClick={toggle}>{isOpen ? 'Close' : 'Ask a Question'}</Button>}
    />
  );
}
```

### With Suggested Prompts

```tsx
import { AskAI } from 'peam/client';

export default function Page() {
  return (
    <AskAI
      type="dialog"
      suggestedPrompts={['How do I get started?', 'What models are supported?', 'How do I deploy?']}
    />
  );
}
```

## Props

| Prop               | Type                               | Default       | Description                                          |
| ------------------ | ---------------------------------- | ------------- | ---------------------------------------------------- |
| `type`             | `'chat' \| 'dialog' \| 'sidepane'` | `'chat'`      | Display mode for the chat interface                  |
| `endpoint`         | `string`                           | `'/api/peam'` | API endpoint used by the chat transport              |
| `suggestedPrompts` | `string[]`                         | `undefined`   | Suggested prompts to display to the user             |
| `maxMessages`      | `number`                           | `10`          | Max messages before summarization (see Notes)        |
| `inlineButton`     | `boolean`                          | `false`       | Render the default button inline instead of floating |
| `buttonVariant`    | `'icon' \| 'iconLabel'`            | `'iconLabel'` | Variant for the default button                       |
| `className`        | `string`                           | `undefined`   | Additional CSS classes for the root container        |
| `button`           | `function`                         | `undefined`   | Custom button render function                        |

## Notes

- **Dialog mode** opens the chat in a modal overlay.
- **Sidepane mode** slides the chat in from the side.
- **Chat mode** displays the chat inline on the page.
- `buttonVariant` is used by the default `PeamButton` in chat mode.
- `maxMessages` is defined on the props type but is not currently wired into the client components.
