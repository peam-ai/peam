# Overview

Modal dialog chat surface. Pair it with `AskAI.Trigger`.

## Usage

```tsx
import { AskAI } from 'peam/client';

export default function Page() {
  return (
    <AskAI>
      <AskAI.Trigger />
      <AskAI.Dialog />
    </AskAI>
  );
}
```

## Notes

- `AskAI.Dialog` must be rendered inside `AskAI`.
- When `button` is not provided, the default floating `PeamButton` is used.
